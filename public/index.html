Tutoriel

3 étapes de préparation, 5 minutes.

Avant d'utiliser le configurateur, préparez ces 3 éléments. Tout est gratuit, aucune carte bancaire requise.

1
📄

Créez votre Google Sheet

Votre FAQ vit dans un Google Sheet. 3 colonnes obligatoires + 1 optionnelle.

  1. Ouvrez un nouveau Google Sheet : sheets.new ↗
  2. Dans le nouveau Sheet, cliquez la cellule A1 et faites Ctrl + V (ou ⌘ + V sur Mac). Les 4 en-têtes apparaissent automatiquement :
    thème question réponse url
    La 4e colonne url est optionnelle : si remplie, le bot ajoutera un bouton « 👉 En savoir plus » qui pointe vers cette URL à la fin de sa réponse.
  3. Remplissez les lignes suivantes avec vos Q/R (1 question par ligne). La colonne url peut être laissée vide ou pointer vers un article de blog, une page produit, etc.
  4. Cliquez sur Partager (en haut à droite du Sheet).
  5. Sous Accès général : « Tout le monde avec le lien » → rôle « Lecteur ».
  6. Cliquez Copier le lien et collez-le dans le configurateur ci-dessous.
💡 Astuce : Vous pourrez modifier votre Sheet à tout moment, le bot voit la nouvelle version dans les 5 minutes.
👁️ Voir un Sheet d'exemple 15 Q/R prêtes à l'emploi · Fichier → Créer une copie ↗
2
🔑

Récupérez votre clé Gemini

Gratuit jusqu'à 500 requêtes/jour. Aucune carte bancaire requise.

  1. Allez sur Google AI Studio : aistudio.google.com/app/apikey ↗
  2. Connectez-vous avec votre compte Google.
  3. Cliquez sur « Create API key » (en haut à droite).
  4. Sélectionnez ou créez un projet Google Cloud.
  5. Copiez la clé qui commence par AIzaSy... et gardez-la précieusement.
⚠️ Sécurité : Cette clé est sensible. Préférez le mode variable d'environnement Netlify (production) plutôt que le mode clé en brut (démo uniquement, visible côté navigateur).
3
📋

Complétez le formulaire et collez le code

Une fois votre Sheet et votre clé prêts, il ne reste qu'à configurer puis copier-coller le snippet sur votre site.

  1. Descendez jusqu'au configurateur ↓ ci-dessous.
  2. Complétez les 4 sections : identité visuelle, contenu, source de données (collez le lien de votre Google Sheet), email de contact.
  3. L'aperçu à droite se met à jour en temps réel - vous voyez exactement ce que verront vos visiteurs.
  4. Cliquez sur Copier en bas pour récupérer le snippet HTML.
  5. Collez ce code dans le HTML de votre site, juste avant la balise </body>, sur toutes les pages où vous voulez afficher le chatbot.
💡 Astuce : Sur WordPress, Shopify ou Wix, il existe un endroit prévu pour insérer du code (ex. « Code personnalisé », « Pied de page », « Custom HTML »). C'est là que vous collez le snippet.

Deux modes pour mettre votre clé Gemini en place :

Recommandé 🔒 Mode production

La clé est stockée en variable d'environnement Netlify (GEMINI_API_KEY). Jamais visible côté navigateur. C'est le mode par défaut, et c'est ce qu'il faut faire en production. Dans le configurateur, laissez la case « Mettre la clé en brut » décochée.

Démo / Test ⚡ Clé en brut dans le snippet

La clé est mise directement dans le HTML via data-api-key="AIzaSy...". Pratique pour un test rapide ou un déploiement perso. À éviter en production : la clé est visible par n'importe quel visiteur via un simple F12.

Configurateur

Composez votre chatbot.

Renseignez vos paramètres, l'aperçu se met à jour en temps réel. Quand c'est bon, copiez le snippet et collez-le sur votre site.

Questions

      
Vos questions, nos réponses.

      
Et bien sûr - vous pouvez tester directement notre propre chatbot en bas à droite de cette page 👋

    

    

      

        Que comprend exactement les 499€ ?
        
L'installation complète clé en main : on prend votre FAQ (en l'état, on s'occupe de la structurer dans un Google Sheet propre), on personnalise le chatbot à vos couleurs, on configure l'IA Gemini pour votre cas d'usage, on l'installe sur votre site, on forme votre équipe en visio, et on assure 1 mois de support après la mise en ligne. À la fin, le code vous appartient.

      

      

        Pourquoi pas d'abonnement mensuel ?
        
Parce qu'on trouve que c'est plus honnête. Le coût pour vous se résume à : 499€ une fois, plus le coût direct de Gemini chez Google (5-30€/mois selon votre trafic, payés directement à Google sans marge de notre part). Vous savez exactement ce que vous payez et à qui.

      

      

        Combien de temps pour avoir mon chatbot en ligne ?
        
5 jours ouvrés à partir du moment où vous nous transmettez vos questions-réponses. Jour 1 : structuration de votre FAQ dans Google Sheets. Jour 2-3 : configuration et personnalisation. Jour 4 : démo et ajustements. Jour 5 : installation sur votre site et formation.

      

      

        Sur quels CMS / sites est-ce que ça fonctionne ?
        
Sur tous : WordPress, Shopify, Wix, Squarespace, Webflow, Joomla, sites HTML statiques, sites React/Next.js, etc. Le seul prérequis est de pouvoir insérer une balise <script> dans votre HTML, ce que tous les CMS permettent.

      

      

        Comment ma FAQ est-elle alimentée ?
        
Vous créez un Google Sheet avec 3 colonnes obligatoires (thème, question, réponse) et 1 colonne optionnelle (url, qui ajoute un bouton « En savoir plus » à la réponse du bot). Vous le partagez en lecture publique (« Tout le monde avec le lien » → rôle « Lecteur »). Pour les mises à jour : vous éditez le Sheet comme d'habitude, le bot voit la nouvelle version dans les 5 minutes. Pas d'interface tierce à apprendre. Le tutoriel complet est en haut de cette page.

      

      

        Que se passe-t-il si le bot ne sait pas répondre ?
        
Le bot répond explicitement qu'il n'a pas l'info dans la FAQ et invite le visiteur à vous écrire à l'adresse de contact que vous avez configurée. Le bouton « Nous écrire » est toujours visible en bas du panneau de chat. Si l'API Gemini est en panne ou si votre Sheet est inaccessible, même chose : le bot bascule en mode email avec votre adresse de contact.

      

      

        Le bot peut-il inventer des réponses ?
        
Non. Le système contraint strictement le bot à répondre uniquement à partir de votre Sheet. S'il ne trouve pas l'information, il invite le visiteur à vous contacter par email. Vous gardez le contrôle total du contenu.

      

      

        Mes données sont-elles sécurisées ?
        
Oui. En mode production, votre clé API Gemini est stockée chiffrée côté serveur (variable d'environnement Netlify), jamais exposée dans le navigateur. Les conversations transitent en HTTPS. Pour les besoins RGPD stricts (B2B), nous configurons le tier Gemini payant qui garantit qu'aucune donnée n'est utilisée pour entraîner les modèles Google.

      

      

        Et après le mois de support inclus ?
        
Le système continue de fonctionner sans intervention de notre part. Vous gérez vous-même les mises à jour de votre FAQ via Google Sheets. Si vous voulez un support continu ou des évolutions custom (ex : ajouter des fonctionnalités), nous proposons des packs ponctuels à partir de 99€.

      

      

        Puis-je tester avant de payer ?
        
Bien sûr. Le configurateur ci-dessus vous montre exactement ce que vous obtiendrez visuellement. Pour une démo personnalisée avec votre vraie FAQ, écrivez-nous à contact@supportai.fr - on vous propose un appel de 20 minutes gratuit où on vous montre une version live avec vos données.
